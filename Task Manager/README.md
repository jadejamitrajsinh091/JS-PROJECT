# ✅ JavaScript Project – Task Manager (CRUD)

This project is a Task Manager application built using HTML, Bootstrap, and JavaScript.

It allows users to manage daily tasks by adding, editing, and deleting task cards through Bootstrap modals. The complete UI design was also created by me.

## ▶️ How to Use

1. Click the **Add Task** button.
2. Enter the task title, description, and due date.
3. Click **Add Card** to create a new task.
4. Use the **Edit** button to update task details.
5. Use the **Delete** button to remove a task.
