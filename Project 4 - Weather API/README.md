# 🌦️ JavaScript Project – Weather API Application

This project is a Weather API Application built using HTML, CSS, and JavaScript.

It allows users to search for any city and displays live weather information using the OpenWeather API. The project also includes a custom UI design created by me.

## ⚙️ Features

✅ Search weather by city name  
✅ Live Weather API integration  
✅ Displays temperature in Celsius  
✅ Shows wind speed  
✅ Shows current weather condition  
✅ Error handling for invalid city names  
✅ Responsive user interface  
✅ Custom UI designed by me  

---

## ▶️ How to Use

1. Enter the city name.
2. Click the **Submit** button.
3. Wait for the API response.
4. View the current weather details.

---

## 🖥️ Output

```
City Name
Temperature (°C)
Wind Speed
Weather Condition
```
